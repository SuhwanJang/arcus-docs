## 1.12.2 (2020-09-04)

**New Features**

- 1MB 초과 크기의 아이템 저장
  - 최대 1MB 아이템 크기에 대한 코드 수준의 제약을 제거하였습니다.
  - 따라서, arcus-memcached 구동 옵션인 `-I` 옵션으로 최대 아이템 크기를 늘려서 구동하면, 1MB 초과 크기의 아이템을 저장할 수 있습니다.
